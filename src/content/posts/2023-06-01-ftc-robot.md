## Overview

Designed and programmed a competition-ready robot for the FIRST Tech Challenge (FTC) 2022–2023 season. The robot handled both driver-controlled tele-op and fully autonomous operation modes.

## What I Built

- **Tele-op control system** — mapped controller inputs to a multi-motor drivetrain with smooth acceleration curves
- **Autonomous routines** — sensor-driven path planning using encoders and IMU data to hit scoring targets reliably
- **Sensor integration** — combined distance sensors, color sensors, and the onboard IMU for environment awareness
- **Modular architecture** — split subsystems into separate OpMode classes so teammates could work on mechanisms independently

```java
@Autonomous(name = "AutoDrive", group = "Competition")
public class AutoDrive extends LinearOpMode {
    @Override
    public void runOpMode() {
        waitForStart();
        drive.forward(24); // inches
        arm.extend(ArmPosition.HIGH);
        claw.drop();
    }
}
```
