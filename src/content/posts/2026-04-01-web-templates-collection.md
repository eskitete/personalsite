## Overview

A growing collection of polished, production-ready website templates spanning multiple industries. Each template is built with a modern stack — React, TypeScript, Vite, and Tailwind CSS — and is designed to be clean, responsive, and client-ready out of the box.

## Design Philosophy

Each template is intentionally scoped to a single industry vertical so clients can see themselves in the design immediately. Shared principles across all templates:

- **Mobile-first layouts** — fully responsive from 320px up
- **Component-driven architecture** — easy to swap sections in and out
- **Minimal dependencies** — fast load times out of the box
- **Accessible markup** — semantic HTML throughout
